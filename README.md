# 📊 Business Sales Performance Dashboard (Task 1)

## 🔍 Overview

This project analyzes business sales data using Power BI to identify trends, top-performing products, and regional performance.

## 🛠 Tools Used

* Power BI
* Excel Dataset

## 📈 Key Insights

* Technology category generates the highest profit (~51K)
* West region leads in total sales
* Sales show peak trends over time
* Furniture category has lower profitability

## 💡 Recommendations

* Focus on high-profit categories
* Expand in high-performing regions
* Improve low-performing categories
* Promote top-selling products
